# Security Policy

## Reporting a Vulnerability

If you believe you have found a security issue in Tafrah, please report it privately to:

- `oghie@perkamindo.org`

Please include:

- the affected crate or API surface
- a minimal reproduction when possible
- impact assessment
- platform and toolchain details

## Disclosure Process

- reports will be reviewed privately first
- fixes will be prepared before public disclosure when practical
- release notes will document security-relevant fixes once patches are available

## Scope Notes

Tafrah is a cryptographic library. Reports about correctness, side channels, malformed-input handling, key erasure, and cross-language ABI safety are all in scope.
