# Kotlin Guide

The Kotlin example reuses the JNI bridge from the Java example.

## Quick Start

```sh
make install PREFIX="$PWD/dist/install"
sh examples/auth-demo/run_language_examples.sh
```

Relevant files:

- `examples/auth-demo/kotlin/Main.kt`
- `examples/auth-demo/java/native/tafrah_jni.cpp`
